# Drafting rules and quality controls

This library is separate from `content/treatises/`, which contains substantive
legal authority for retrieval. Put court style manuals, filing conventions, and
other drafting guidance in `style-guides/source/<guide-slug>/`. Put
machine-readable validation policies in `checks/` (for example, citation,
caption, privacy, and local-filing rules).

Sources in this directory may inform drafting and validation, but must be
identified as drafting guidance rather than substantive legal authority when
they are retrieved. Preserve received files and record their version/date in
the file name or companion metadata.

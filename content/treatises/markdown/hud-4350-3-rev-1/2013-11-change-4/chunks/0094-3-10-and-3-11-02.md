Program Eligibility

L. Verifying Information on Immigration Status Owners must verify the validity of documents provided by applicants or tenants.

The personal computer method provides automated status verification when the information is contained in the Alien Status Verification Index (ASVI) database. It also automates the paper secondary verification process, which eliminates in most instances the completion of the paper Form G-845S. If the owner is unable to obtain the results using the automated primary and secondary verification method, the owner must attempt to obtain results using the secondary verification paper process.
1. Primary verification.
a. Owners must conduct primary verification of eligible immigration status only for persons claiming eligible immigration status.
b. Owners must conduct primary verification through the SAVE web- based program, DHS’ automated system. After obtaining an access code, user ID and temporary password from the Multifamily SAVE Administrator at HUD Headquarters (see subparagraph H above), owners can access SAVE with a personal computer at https://www.vis-dhs.com/ or *through the EIV system.*
c. After accessing the ASVI database, the owner enters the required data fields. The personal computer system will display one of the following messages for immigration status confirmation on the screen.
(1) Lawful Permanent Resident
(2) Temporary Resident
(3) Conditional Resident
(4) Asylee
(5) Refugee
(6) Cuban\Haitian Entrant
(7) Conditional Entrant
2. Secondary verification. If the message institute secondary verification is displayed on the screen, the manual verification process must be used.

# a. No Requirement for Code Violation Citation

## Table-of-contents context

- IX. Tenant Claims for Defective

    - 12. Proof Issues – R.C. 5321.04(A)(1) Code Violation
      - a. No Requirement for Code Violation Citation

## Source text

The issuance of a code violation citation by the applicable governmental entity is not a required element of an R.C, 5321.04(A)(1) contract claim (or negligence per se claim). Wilcox v. Tabler, No. 26530, 2013 WL 1324937 (Ohio App., 9th Dist., Apr.

3, 2013). The tenant is simply required to prove that, during the relevant time, the defective condition existed and it violated the applicable code. Id.

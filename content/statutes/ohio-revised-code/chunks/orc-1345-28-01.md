# Ohio Rev. Code § 1345.28 — Deceptive acts or practices.

## Source text

Failure to comply with sections

1345.21

to

1345.27

of the Revised Code constitutes a deceptive act or practice in connection with a consumer transaction in violation of section

1345.02

of the Revised Code.

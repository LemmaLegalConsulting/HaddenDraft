# Ohio Rev. Code § 1309.101 — Short title - section numbering - UCC 9-101.

## Source text

(A) This chapter may be cited as "Uniform Commercial Code, secured transactions."

(B) This chapter uses the numbering system of the national conference of commissioners on uniform state laws. The digits to the right of the decimal point are sequential and not supplemental to any preceding Revised Code section.

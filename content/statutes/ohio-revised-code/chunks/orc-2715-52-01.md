# Ohio Rev. Code § 2715.52 — Action to be dismissed if attachment refused.

## Source text

If the court or judge refuses to grant an order of attachment under section

2715.50

of the Revised Code, the action shall be dismissed, but without prejudice to a future action. In all such actions application for an attachment must be made.

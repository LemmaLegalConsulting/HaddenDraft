# Ohio Rev. Code § 1345.93 — Returned devices.

## Source text

An assistive device returned to a manufacturer in accordance with section

1345.92

of the Revised Code by a consumer or supplier in this state or another state shall not be the subject of another consumer transaction in this state unless the manufacturer fully discloses the defect to the prospective consumer.

# Ohio Rev. Code § 2329.69 — Exemptions apply to all courts.

## Source text

Sections

2329.63

to

2329.71

of the Revised Code, with respect to exemptions, apply to all courts in this state, and a person is entitled to all applicable exemptions in any case or proceeding before any court or officer.

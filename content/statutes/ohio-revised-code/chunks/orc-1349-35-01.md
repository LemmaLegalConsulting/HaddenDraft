# Ohio Rev. Code § 1349.35 — Annual report.

## Source text

The superintendent of financial institutions shall include, as part of the annual report required by section

1181.09

of the Revised Code, the number of complaints received, the number of enforcement actions taken, and any other relevant data regarding covered loans.

# Ohio Rev. Code § 1345.95 — Remedies waiver of rights.

## Source text

The remedies in sections

1345.90

to

1345.95

of the Revised Code are in addition to remedies otherwise available for the same conduct under state or local law.

Any waiver of rights by a consumer of the provisions of sections

1345.90

to

1345.95

of the Revised Code is void.

# Ohio Rev. Code § 1322.51 — Effect of child support default on registration or license.

On receipt of a notice pursuant to section

3123.43

of the Revised Code, the division of financial institutions shall comply with sections

3123.41

to

3123.50

of the Revised Code and any applicable rules adopted under section

3123.63

of the Revised Code with respect to a certificate or license issued pursuant to this chapter.

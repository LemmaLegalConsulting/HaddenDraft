# Ohio Rev. Code § 2308.031 — Use of plywood prohibited.

(A) No person shall use plywood to secure real property that is deemed vacant and abandoned under section

2308.02

of the Revised Code.

(B) Division (A) of this section shall not apply to any person that uses plywood to secure real property that is deemed vacant and abandoned under section

2308.02

of the Revised Code prior to the effective date of this section .
